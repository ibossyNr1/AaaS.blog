# Language-Specific Debugging Tools

### JavaScript/TypeScript
- **debugger**: Pause execution in browser/Node.
- **console.table/time/trace**: Advanced logging.
- **Performance marks**: `performance.mark('start')`.
- **VS Code configs**: `launch.json` for Node/Jest.

### Python
- **pdb / breakpoint()**: Built-in debuggers.
- **ipdb**: Enhanced interactive debugger.
- **cProfile**: For performance bottlenecks.
- **logging**: `logging.basicConfig(level=logging.DEBUG)`.

### Go
- **Delve (dlv)**: Command line debugger.
- **debug.PrintStack()**: Immediate trace.
- **pprof**: CPU and memory profiling.
